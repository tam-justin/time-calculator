Time Calculator
A simple calculator allowing you to perform calculations on time!
You can use this input to calculate the total length of time given by numerous strings of time.
As a reminder, you must format your input using "XX:XX:XX" including both digits in each section. If any portion of the time string is not needed, use a 0 as a placeholder.
Any invalid inputs will not be added to the total.
